merge¶

"""pd.merge(left,right,on='column') # on used to merge on given column pd.merge(left,right,on='column',how='inner'or'outer'or'right'or'left') # how will be used for different merge in pandas pd.merge(left,right,on='column',how='inner',indicator=True) # indicator are used to show row from which table pd.merge(left,right,on='column',how='inner',indicator=True,suffixes=('left suffixes_name','right suffixes_name')""" #suffixes is used add name to the same columns in both the table
